# PWA Icon Generator Instructions

To generate PWA icons, you can use online tools or create them manually:

## Required Icons:
1. `pwa-192x192.png` - 192x192 pixels
2. `pwa-512x512.png` - 512x512 pixels
3. `apple-touch-icon.png` - 180x180 pixels
4. `favicon-32x32.png` - 32x32 pixels
5. `favicon-16x16.png` - 16x16 pixels
6. `safari-pinned-tab.svg` - SVG format

## Quick Generation:
1. Create a base icon with Ghana colors (Red, Yellow, Green) and "H" or "HG" logo
2. Use online tools like:
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator
   - https://favicon.io/

## Design Guidelines:
- Use Ghana flag colors: Red (#CE1126), Yellow (#FACC15), Green (#006B3F)
- Include "H" or "HG" for HandyGhana
- Ensure icons are clear and recognizable at small sizes
- Use transparent background for PNGs
- SVG should be monochrome for Safari pinned tab
